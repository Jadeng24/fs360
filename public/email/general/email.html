8-Week Node / React / Redux Bootcamp


Picking the right bootcamp can be a difficult decision. In addition to cost, job placement, and location, students have to consider the actual technology being taught. Most bootcamps teach Ruby on Rails or MEAN Stack which is certainly valuable but not very forward-thinking. RoR has been around for over a decade and Angular 1 is migrating away to Angular 2 - which bootcamps do not teach. Much more importantly, the web in general is moving away from both of those and towards Node with React and Redux.

Rather than joining the mass of Rails or MEAN Stack junior devs who graduate from bootcamps every few months, learning the skills which will be high in demand tomorrow is a prudent decision. The Velocity 360 8-Week Fundamentals Bootcamp focuses on these areas:

Topics

✓ Node, Express, Mongo DB
✓ REST APIs
✓ React
✓ Redux
✓ Testing in React
✓ React Router
✓ Server Side Rendering
✓ React Native (for mobile)
By the end of the course, students will have a solid foundation and be able to build their own React/Redux projects from scratch and incoporate a mobile product with React Native.
