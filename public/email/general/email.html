Starting With React and Redux


Getting started with React and Redux can be confusing, especially for beginners. Despite this, there is no better time to start learning React/Redux as it is has firmly established itself as a major player among top echelon of web dev tools - Ruby on Rails, Django, Node JS, etc.

In this online workshop, we will set up a basic React app with Redux integration while covering the concepts behind each step. In addition, we will create a basic Node API in order to provide a realistic source of data for the React project. This workshop is designed for beginners with rudimentary JavaScript knowledge.

Topics

✓ Node
✓ Express
✓ REST API
✓ React
✓ Redux
