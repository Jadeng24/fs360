Intro to Node JS


Join us for a Node JS demonstration where we create and deploy a full web application (using MongoDB, Express, and Node JS) to Heroku at "live" pace - almost as fast as for an actual project. Along the way, I will explain the concepts and reasoning behind each step. We will also explore front end frameworks such as Angular JS and React. This is a great intro for beginners and those considering software development as a career.
